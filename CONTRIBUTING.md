# Contributing

Thanks for helping improve Film Color Script Generator.

## Local Setup

1. Install Node.js 22 or newer.
2. Run `npm install`.
3. Run `make install-hooks`.
4. Run `make dev`.

## Commit Style

Use Conventional Commits:

- `feat: add export controls`
- `fix: handle empty video duration`
- `docs: explain Pages deployment`
- `test: cover palette clustering`

Run `make test`, `make build`, and `make smoke` before pushing.
